Variables d’environnement

Le serveur lit les paramètres suivants depuis le fichier .env (ou les variables d’environnement du système) :

HOST=0.0.0.0          # Adresse d’écoute du serveur Flask
PORT=5000             # Port TCP du serveur
GYRO_SAMPLE_RATE=0.05 # Intervalle de lecture du capteur en secondes (20 Hz)

Ces variables sont importées dans Robot/config.py (à créer) afin de centraliser la configuration :

# Robot/config.py
import os

HOST = os.getenv("HOST", "0.0.0.0")
PORT = int(os.getenv("PORT", "5000"))
GYRO_SAMPLE_RATE = float(os.getenv("GYRO_SAMPLE_RATE", "0.05"))

Gestion du logging

Un fichier de configuration JSON/YAML pour le module logging permet de contrôler le niveau et le format sans modifier le code :

# logging_config.json
{
  "version": 1,
  "formatters": {
    "standard": {"format": "% (asctime)s %(levelname)s %(name)s %(message)s"}
  },
  "handlers": {
    "console": {"class": "logging.StreamHandler", "formatter": "standard", "level": "INFO"}
  },
  "root": {"handlers": ["console"], "level": "INFO"}
}

Dans le code :

import logging, json
logging.config.dictConfig(json.load(open('logging_config.json')))
logger = logging.getLogger(__name__)
