Vue d’ensemble

Ce projet fournit une petite application web Flask + Socket.IO qui lit les données d’un capteur inertiel MPU‑6050 et les diffuse en temps réel à un navigateur.

La documentation suivante détaille l’architecture, la configuration, le déploiement et les bonnes pratiques pour faire évoluer le projet.

Commencer rapidement

# créer l’environnement virtuel
python -m venv .venv
.venv\Scripts\activate  # sous Windows
# installer les dépendances
pip install -r requirements.txt
# lancer le serveur
python -m Robot.server
